## 验证执行顺序

1. 先执行 N01-N03（正常场景）确认基础功能
2. 执行 C01-C03（舒适流程）验证状态管理
3. 执行 B01-B04（边界场景）验证 A2 约束力
4. 执行 F01-F03（承诺跟进）验证 time node
5. 执行 X01-X03（跨日跟进）验证完整行为链
6. 执行 R01-R04（显式回忆）验证检索质量
7. 执行 S01-S03（关系转变）验证状态延续
