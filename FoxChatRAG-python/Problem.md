# 以下为项目过程中遇到的问题
## 一个陪伴者需要有哪些方面
由于我的陪伴模型只是为了一个机器人，我不需要一个模仿的“人”
我想做的是一个会说你想听话的机器，
让用户知道，对话的就是一个机器人，不会有最近的事，不会有最近的爱好，就像梦里所说的，“她一直都这样”
让用户明白，对话的这个人不是一个真的人，只是一个说话像自己思念的那个人一样的机器
